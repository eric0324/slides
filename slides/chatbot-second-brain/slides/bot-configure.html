STEP 02 · CLAUDE CODE

把 token 跟 access 設定好。

~/vault — claude
~/vault $ claude
> /telegram:configure
  paste your bot token:
  ▸ 7891234567:AAH-xyZ...vbn
  ✓ token saved

# 把自己加進 allowlist
> /telegram:access add 12345678
  ✓ user 12345678 allowed
依序執行

02

03
再跑 /telegram:access add <id> — 加 allowlist

在 Telegram 打開你的 bot，
